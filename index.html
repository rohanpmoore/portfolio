End-of-class Projects

  1. Welcome to Epicodus: The first project I completed for the end of day code review, we made a cheat sheet for some of the things we had learned the first day of class, with links to important places like learntocode, epicodus attendance, and stack overflow. It was created entirely with HTML.
  2. Resort Website: The second day's code review project, a webpage encouraging people to come to a made up resort island. It includes several webpages hyperlinked to each other and some basic CSS styling, along with HTML for the base code.
  3. Animal Shelter: The third day's code review project, this is a website for a fake animal shelter listing several animals they have available for adoption. It makes greater use of CSS styling and has a responsive setup to adapt to different window sizes, along with HTML for the base code.
  4. Wiki Clone: The fourth day's code review project, it is our best effort at recreating the wikipedia homepage. It makes significant use of Bootstrap, especially for the grid layout options, alongside custom CSS and HTML.
